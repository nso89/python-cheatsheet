#### <a name="using-datetime.now()-with-strftime()"></a>Using `datetime.now()` with `strftime()`:
```python
print(datetime.now().strftime("%Y%m%d-%H%M%S"))
```
#### <a name="using-time.strftime()"></a>Using `time.strftime()`:
```python
print(time.strftime("%Y%m%d-%H%M%S"))
```
